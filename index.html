DICE ROLLER

This is a page layout exhibiting clean, bright design principles using HTML, CSS, JavaScript, and Jquery. There are multiple die choices and a selectable number of rolls. It is meant to be used as a supplemental to any tabletop games where dice are used.

WEATHER MAP

The Weather Map is a Javascript API project to create a user interactive application where the user can input location, latitude or longitude, or drag a marker on the map to show the realtime weather forecast in a given location through Dark Sky API and MapBox API.

GARDENSEEDR

GardenSeedr is a desktop gardening assistant developed during the coronavirus outbreak as an aid to new gardeners with planting ideas and placement. Two separate API calls, OpenWeather API and OpenFarm API were used for weather updates and crop information respectively. A virtual garden grid is used for crop placement.

GRADIENT MIND

Gradient Mind is an experiment with pure CSS and HTML showcasing some interesting animation, layout and design.
